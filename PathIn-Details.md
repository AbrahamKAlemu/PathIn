# Hackathon Project Summary

## Project Idea
This project is a LinkedIn-style career exploration and recommendation tool for Gen-Z and Gen-Alpha users. It turns a user’s profile, school history, and job history into a guided career simulation that shows possible paths, likely next roles, and relevant opportunities.

## What It Does
The flow starts with a user profile and uses that information to generate a career path preview. From there, the system recommends scholarships, roles, clubs, courses, and other opportunities that fit the user’s current standing and experience. It can also compare the user with similar profiles to show how people with similar backgrounds acquired their first roles, later positions, and achievements.

## Why It Fits the Hackathon
The project matches the hackathon challenge of making LinkedIn more valuable to Gen-Z and Gen-Alpha users by improving career discovery and early-career planning [file:2]. It also fits the Career Launchpad track because it helps users understand their market value, identify skill gaps, and see a clearer route to their first role [file:2].

## Core Features
- Profile-based career simulation.
- Career path preview using similar-user histories.
- Personalized suggestions for scholarships, roles, clubs, and courses.
- Progression view showing current standing, suggested positions, and long-term paths.
- Optional AI-powered guidance to make recommendations feel tailored and interactive.

## Data Inputs
The starter dataset includes user data with school history, job history, current location, posts/activity, skills, and courses, plus job and course datasets [file:2]. The project can also use free public APIs, but it should avoid extra proprietary data scraping [file:2].

## Value Proposition
The main value is helping students move from “Where am I now?” to “What should I do next?” in a practical, personalized way. It also adds transparency by showing how similar users built their careers and what steps led to each position.

## Demo Story
A good demo can show a student profile, generate a career path, and then display recommendations with explanations. The story should highlight the user need, the suggested path, and the impact of making career planning feel more actionable and less overwhelming.
